# djblog project
---

## how to launch a project

+ склонировать репозиторий
  ```
  git clone https://github.com/toth3m00n/djblog
  ```
  
+ перейти в папку проекта djblog_root
  ```
  cd djblog_root
  ```
  
+ установить все зависимости 
  ```commandline
  python -m pip install -r requirements.txt
  ```
